<HTML>
-글을 감싸는 틀 = 태그, 틀에 붙는 부가 설명 = 속성

HTML 코드
1. “이거 HTML로 작성된 문서야~”를 알려주는 태그
2. 직접 화면에 등장하진 않지만 이 문서를 설명하는 태그
예) 인코딩 방식
3. 직접적으로 화면에 등장하는 문서에서 보이는 태그
- <h1><h2><h3>..</h1></h2></h3> : 제목을 입력할 때 사용 (h1부터 큰 글자)
- <p></p> :　본문 단락 표시
- <br> : 줄 바꿈
- <div></div> :　별도의 단락 표시
- <a href=“링크 경로”></a> : 하이퍼링크 삽입
- <ul> : 순서가 없는 목록 표시
- <ol> : 순서가 있는 목록 표시
- <li> : 목록 구성의 요소를 표시
